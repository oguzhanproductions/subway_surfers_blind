Place licensed music files in this folder using one of these slot names:

- `menu_intro.ogg`, `game_intro.ogg`, `menu_theme.ogg`, or `menu.ogg`
- `gameplay_main.ogg`, `subway_surfers_theme.ogg`, `main_theme.ogg`, `theme.ogg`, or `run_theme.ogg`

The game resolves the first matching file for each slot and applies fade-in, fade-out, and menu-to-gameplay switching automatically.

Research notes:

- Official Subway Surfers music is distributed through the official Subway Surfers artist catalog on Apple Music and through the in-game Boombox/Spotify flow mentioned by SYBO.
- `Game Intro` is the safest menu-slot candidate based on its official track name.
- `Subway Surfers Theme` is the safest gameplay-slot candidate based on its official name and Boombox usage.
- Seasonal city tracks should only be added if you have a licensed copy and a clear mapping for the version you want to emulate.
