Subway Surfers Blind Edition
Changelog

Date: 2026-03-30
Version: 1.1.4

Update Summary
- Updated the game version to 1.1.4.
- Added a weekly seasonal leaderboard system with automatic season rollover, season countdowns, and season-specific rewards.
- Added fixed custom season names for every 2026 leaderboard week through the end of the year.
- Fixed mission and quest progress tracking so progression keeps updating even when quest announcements are disabled.
- Added a daily reset flow inside Quests so players can clear today's daily quest, Word Hunt, Daily High Score, and Coin Meter progress without reopening already claimed rewards.
- Added a dedicated announcer callout system for critical gameplay prompts, with bundled voice clips that can also be previewed from Learn Game Sounds.

Gameplay Changes
- Leaderboard rankings now reset at the end of each weekly season and start the next season from a clean board.
- Season finalization now preserves account rewards in a separate inbox while clearing expired leaderboard submissions, season rows, and reward rows so only active-season leaderboard data remains on the server.
- Seasonal rewards now scale across ranks 1 through 10 instead of rewarding only the top and bottom positions, with item seasons awarding 5, 5, 4, 4, 3, 3, 2, 2, 1, and 1 items by rank.
- Coin reward seasons now scale from 3x run coins for rank 1 down to 1x run coins for rank 10 based on each winner's published run coin total.
- Hoverboards now work at most 4 times in a single run, even if the player owns more boards in inventory.
- Key revives now work at most 3 times in a single run, and further crashes end the run even if the player still owns more keys.
- Mission completion progress now continues to count in the background regardless of whether quest announcement speech is enabled.
- Completed mission rows now cap their visible progress at the target value so the Missions menu reports finished goals cleanly.
- Daily reset keeps claimed quest, Word Hunt, Daily High Score, and Coin Meter rewards locked while clearing the resettable daily counters.

Audio or UI Updates
- Leaderboard now opens without the spoken "leaderboard loaded" announcement and instead shows the current season name, countdown, and reward preview directly in the menu.
- Leaderboard help and menu labels now reflect the weekly seasonal board flow instead of the old multi-period ranking system.
- Restored the leaderboard connection and new personal best publish sounds so successful server connections and published high scores play their intended audio feedback again.
- Renamed the Gameplay Announcements option from Quest Changes to Quest Announcements so the setting matches its real behavior.
- Added a Reset Today's Progress action to the Quests menu with spoken confirmation that claimed rewards remain preserved.
- Replaced the gameplay obstacle NVDA callouts with the new bundled announcer prompt clips while keeping train fly-by warnings and speed-based early timing logic.
- Added the new announcer prompt clips to Learn Game Sounds so players can preview the jump, roll, move left, and move right callouts from the menu.

Date: 2026-03-22
Version: 1.1.3

Update Summary
- Updated the game version to 1.1.3.
- Added a secure online leaderboard system with account sign-in, score publishing, player profiles, and persistent session support.
- Expanded the leaderboard with all-time, daily, weekly, and monthly ranking windows plus difficulty-specific filtering for Easy, Normal, and Hard runs.
- Added richer published run records with surfaced game version data, distance, death reason, clean escapes, revive usage, power-up usage, and leaderboard verification status.
- Added profile-level performance summaries including recent run averages, best score improvement, total published runs, and active-day tracking.
- Added an original-style Events hub with Daily High Score, Coin Meter, Mini Mystery Box, Daily Login Calendar, and surfaced Word Hunt and Season Hunt status.
- Added a separate Missions hub that now splits mission sets, quests, and achievements into dedicated menu flows.
- Added a Me hub with direct access to Characters, Boards, Item Upgrades, and Collections.
- Added persistent Boards with active board selection and original-style board powers including Double Jump, Super Jump, Super Speed, Smooth Drift, Zap Sideways, and Stay Low.
- Added persistent Collections with completion tracking and passive reward bonuses tied to character and board sets.
- Added a separate Quests system with daily and seasonal quest generation, sneaker rewards, and a sneaker meter reward track.
- Added daily event scheduling inspired by the original game with Super Mysterizer, Mega Jackpot, Character Bonus, Super Mystery Box Mania, and Wordy Weekend rotations.
- Added an original-style Item Upgrades submenu inside the Shop for Coin Magnet, Jetpack, 2X Multiplier, and Super Sneakers.
- Added persistent per-item upgrade levels with escalating coin costs and immediate in-menu feedback after each purchase.
- Added a dedicated main-menu description system with English guidance for each main menu action.
- Added a configurable desktop exit confirmation flow for the main menu Exit action and Escape shortcut.
- Added an optional pause-on-focus-loss setting so active runs automatically pause when the game window loses focus.

Gameplay Changes
- Leaderboard submissions now retain richer run telemetry so competitive comparisons can include more than raw score and coins.
- Added basic run verification heuristics that flag suspicious leaderboard entries when submitted run data is internally inconsistent.
- Daily events now affect active gameplay, including featured-character multiplier bonuses, boosted Super Mystery Box appearances, jackpot-biased mystery rewards, and stronger weekend Word Hunt behavior.
- Word Hunt now supports event-driven target words so Wordy Weekend can use the selected character name instead of the standard daily word rotation.
- Daily High Score now tracks the best run of the day and unlocks tiered rewards through the Events hub.
- Coin Meter now tracks daily coin collection toward separate reward tiers.
- Daily quests and seasonal quests now progress from run actions, support claims, and grant sneakers for the new quest meter.
- Hoverboard activation now uses the selected board and applies board-specific powers during the run.
- Collections now grant passive gameplay bonuses such as stronger end-of-run banking, longer hoverboard duration, longer temporary power-up duration, and a higher starting multiplier.
- Coin Magnet upgrades now extend magnet pickup duration through five persistent shop levels.
- Jetpack upgrades now extend flight duration through five persistent shop levels.
- 2X Multiplier upgrades now extend score multiplier pickup duration through five persistent shop levels.
- Super Sneakers upgrades now extend jump-boost duration through five persistent shop levels.
- Headstart, Score Booster, Hoverboard stock, Pogo Stick, Keys, and Mystery Boxes remain outside the item-upgrade ladder so the shop matches the intended original-style system more closely.

Audio or UI Updates
- Added leaderboard prompts and publishing flows for game over and run exit back-to-menu transitions.
- Added in-menu leaderboard filter controls for ranking period and difficulty without leaving the leaderboard screen.
- Expanded profile and run-detail views to announce game version, difficulty, verification state, review notes, and richer run statistics.
- Reworked the main menu surface to add dedicated Events, Missions, and Me destinations instead of keeping every progression system flat on the front page.
- Added dedicated Boards and Collections menus with spoken summaries, active-state announcements, and detailed status rows.
- Added dedicated Events and Quests menu screens with reward claim actions and live progress readouts.
- Added a free daily gift entry inside the Shop so the mini mystery box flow is reachable from the original economy surface.
- Improved the run intro by keeping intro_start on every run while reserving intro_shake and intro_spray for active headstarts only.
- Fixed collision audio so each obstacle type now triggers its intended hit sound instead of falling back to the same generic crash layer.
- Fixed the mono audio routing bug so reward and interface sounds are no longer forced through the HRTF mono path while menu feedback cues stay mono.
- Added an Item Upgrades shop submenu with list and detail screens, level readouts, duration readouts, max-level handling, and purchase announcements.
- Added optional spoken and on-screen descriptions for main menu items, with a direct toggle in Options.
- Added a gameplay T hotkey to announce the current play time on demand.
- Added an Exit Confirmation option in Options so players can choose whether the game asks before returning to desktop.
- Added a Gameplay Announcements toggle that lets players enable or disable automatic pausing when the game window loses focus.
- Added line-by-line clipboard copying for How to Play and What's New, plus a Copy All action at the end of each list.
