Subway Surfers Blind Edition
Changelog

Date: 2026-03-22
Version: 1.1.3

Update Summary
- Updated the game version to 1.1.3.
- Added a persistent Character Upgrades section inside the Shop with unlock, active character selection, and per-character leveling.
- Expanded the roster to the officially referenced classic crew: Jake, Tricky, Fresh, Yutani, Spike, Dino, and Boombot.
- Moved SAPI controls into a dedicated submenu under Options and added direct SAPI volume control.
- Reworked updater installation so release ZIP contents are staged, applied on restart, and cleaned up automatically after install.
- Added an original-style Item Upgrades submenu inside the Shop for Coin Magnet, Jetpack, 2X Multiplier, and Super Sneakers.
- Added persistent per-item upgrade levels with escalating coin costs and immediate in-menu feedback after each purchase.
- Cleaned the changelog structure after the previous merge conflict markers.

Gameplay Changes
- Jake now increases the number of coins banked at the end of a run as he levels up.
- Tricky now extends hoverboard shield duration when activated.
- Fresh now extends temporary power-up duration for jetpacks, magnets, double score, and super sneakers.
- Yutani now starts runs with a higher multiplier as she levels up.
- Spike now adds a stronger coin banking bonus.
- Dino now extends hoverboard protection even further.
- Boombot now extends temporary power-up duration with a stronger late-game bonus.
- Coin Magnet upgrades now extend magnet pickup duration through five persistent shop levels.
- Jetpack upgrades now extend flight duration through five persistent shop levels.
- 2X Multiplier upgrades now extend score multiplier pickup duration through five persistent shop levels.
- Super Sneakers upgrades now extend jump-boost duration through five persistent shop levels.
- Headstart, Score Booster, Hoverboard stock, Pogo Stick, Keys, and Mystery Boxes remain outside the item-upgrade ladder so the shop matches the intended original-style system more closely.

Audio or UI Updates
- Added accessible spoken feedback for character lock state, current level, perk summary, unlock purchases, selection changes, and max-level handling.
- Added current active character status to the Shop and Character Upgrades menus.
- Added a dedicated SAPI Settings submenu with On or Off, volume, voice, rate, and pitch controls.
- Moved gameplay announcement toggles into a dedicated submenu under Options so runtime speech preferences stay grouped together.
- Hardened settings persistence so item upgrade purchases now survive interrupted writes and recover cleanly from a corrupted primary save file.
- Fixed the updater so packaged builds now apply extracted files into the game folder on restart and clean leftover update ZIP files.
- Fixed collision audio so each obstacle type now triggers its intended hit sound instead of falling back to the same generic crash layer.
- Fixed the mono audio routing bug so reward and interface sounds are no longer forced through the HRTF mono path while menu feedback cues stay mono.
- Added an Item Upgrades shop submenu with list and detail screens, level readouts, duration readouts, max-level handling, and purchase announcements.
- Updated the in-game What's New and help copy to mention the new item upgrade system and the audio routing fix.

- Defaulted those progress-related options to off so players can enable only the information they want.

Gameplay Changes
- Coin counters can now be disabled without removing coin spawns from the run.
- Quest changes can now be disabled to stop mission, Word Hunt, and Season Hunt progression updates independently.

Audio or UI Updates

Date: 2026-03-12

Update Summary
- Improved HRTF and spatial audio so obstacle placement feels more believable in 3D space.
- Refined hazard cues with clearer left-right spread, depth, and obstacle height positioning.
- Added direction-aware hazard velocity so approaching warnings sound more natural before and after passing.

Gameplay Changes
- Balanced difficulty more clearly across Easy, Normal, and Hard.
- Easy mode now delays harder train setups and leaves more space between hazards.
- Hard mode now introduces tighter pressure and earlier hazard density than Normal.
- Fixed the lane escape bug by clamping the player back onto the valid three-lane track.
- Adjusted coin and support spawning so pickups can appear more naturally in front of the player instead of favoring the center lane.

Audio or UI Updates
- Fixed the game-over and return-to-menu music flow.
- Menu music now starts as soon as the game-over menu opens.
- Returning to the main menu after a run now keeps menu music consistent instead of waiting for another menu interaction.
- Made the game window resizable.
- Enforced a minimum window size to keep menus and gameplay readable.
